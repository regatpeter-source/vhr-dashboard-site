# Voice Models Data

This directory contains voice recognition and text-to-speech models for the VHR (Voix vers Casque) feature.

## Contents

- Voice recognition models
- Text-to-speech synthesis models
- Language packs for French support
- Audio preprocessing utilities

## Installation

When downloading voice data from the VHR Dashboard:
1. Download the `voice-data.zip` file
2. Extract it to your Meta Quest device
3. Use the "Voix PC→Casque" feature to send voice messages

## Supported Languages

- French (Français) - Main language
- English - Secondary support

## File Structure

```
voice-models/
├── recognition/
│   ├── fr-FR/
│   └── en-US/
├── synthesis/
│   ├── fr-FR/
│   └── en-US/
└── config.json
```

## Notes

- Models are automatically updated through the dashboard
- Requires Meta Quest 2 or newer
- Approximately 500MB total size
- Internet connection required for first-time setup
