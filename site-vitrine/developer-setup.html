Guide : activer le mode développeur & connecter votre casque (USB, puis Wi-Fi)

Ce guide vous accompagne pas à pas pour préparer un casque VR pour fonctionner avec VHR DASHBOARD : activer le mode développeur, effectuer la première connexion par USB, puis activer ADB over Wi‑Fi pour les sessions ultérieures.

Prérequis

  • Un PC (Windows, macOS ou Linux) avec le SDK Platform Tools (adb) installé. Téléchargement officiel : Android Platform Tools.
  • Le casque VR (Meta Quest / Pico / autre casque Android) et le câble USB pour le connecter au PC.
  • L'application mobile ou le compte développeur (pour certains casques comme Meta Quest, il faut créer une organisation sur developer.oculus.com et activer le mode développeur dans l'app mobile).
  • Le fichier d'installation (APK) fourni lors du téléchargement de la démo : downloads/VR-Manager-demo.zip.

Étapes rapides

  1. Installer adb

    Téléchargez Android Platform Tools (adb) et ajoutez le dossier à votre PATH. Vérifiez :

    adb version
adb devices

  2. Activer le mode développeur

    Activez le mode développeur depuis l'app mobile du casque et acceptez le débogage USB à la première connexion.

  3. Install & test

    Installez la démo si nécessaire : adb install -r path\to\VR-Manager-demo.apk

    Vérifiez la présence : adb devices

  4. Auto‑WiFi (nouvelle fonctionnalité)

    Si votre serveur VHR DASHBOARD a l'option Auto‑WiFi activée (via AUTO_WIFI_FROM_USB), il peut automatiquement activer adb tcpip 5555 et tenter une connexion Wi‑Fi lorsque le casque est détecté en USB. Utilisez les commandes habituelles si vous devez le faire manuellement :

    adb tcpip 5555
adb shell ip addr show wlan0
adb connect 192.168.X.Y:5555
adb devices

    Vous pouvez aussi appeler le endpoint serveur :

    POST /api/adb/wifi-connect (body: { serial: 'deviceSerial' })

Pour tout problème, utilisez la page Contact ou consultez la FAQ.

Utiliser VHR DASHBOARD

Après avoir mis en place ADB via USB ou Wi‑Fi, lancez VHR DASHBOARD sur votre PC et utilisez la fonctionnalité de détection des appareils pour lister et gérer les casques. Pour tout problème de connexion, consultez notre FAQ ou contactez le support via la page Contact.
